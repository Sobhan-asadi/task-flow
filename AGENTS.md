# Task Flow

A modern productivity workspace for managing projects, tasks, and daily workflows.

Task Flow is a full-stack application built with modern web technologies, focusing on clean architecture, responsive design, and a scalable project structure.

## Features

- Responsive dashboard interface
- Light and dark mode support
- Multi-language support (English / Persian)
- Modern component-based architecture
- Reusable UI components
- Project and task management workflow
- Clean and maintainable code structure

## Tech Stack

### Frontend

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Motion animations
- next-intl for internationalization
- next-themes for theme management

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose

_(Backend implementation is planned for future development.)_
